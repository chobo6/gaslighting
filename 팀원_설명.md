핵심 단어는 **자동화(automation)** 입니다. 사람이 버튼을 누르는 게 아니라 시스템이 스스로 판단·작동합니다.

## 2. 왜 방향을 바꿨나? (제일 중요)

### 기존 계획 (데이터 분석 중심)
"센서 데이터로 머신러닝(ML) 모델을 학습시켜 위험률을 예측한다"가 핵심이었습니다.

### 무엇이 문제였나
가스 위험은 **이미 정답이 정해져 있는 문제**입니다. 농도가 LEL(폭발 하한계) 같은 기준을 넘으면 위험 — 이건 법으로 정해진 **물리 상수**예요.

그런데 우리 설계는 이렇게 돌아갑니다:

```
사람이 규칙을 만든다 (농도+증가율+지속시간 → 위험률)
        ↓
그 규칙으로 정답(라벨)을 만든다
        ↓
ML이 그 정답을 다시 학습한다   ← 결국 규칙을 베끼는 것뿐!
```

즉, **규칙으로 정답을 만들고 ML이 그 규칙을 다시 배우는 "순환 학습"** 이라 ML이 새로 만들어내는 가치가 거의 없습니다. 그냥 규칙을 직접 쓰는 게 더 빠르고 정확해요.

### 그래서 결론
- **ML은 "위험률 숫자를 내려주는 얇은 API"로 최소화** (RandomForest 1개만, 1주 안에 끝)
- **진짜 가치인 "자동 대응(감지→차단→복구)"을 프로젝트의 주인공으로** 만든다
- **Unity 자동화**에 팀 역량을 집중한다

## 3. 우리가 만드는 자동화 흐름 (폐루프)

"폐루프(closed-loop)"란 **시작→끝이 다시 처음으로 돌아오며 계속 도는 고리**를 말합니다.

```
[무인 자동 데모 시작 — 버튼 1번]
        ↓
① 센서값 재생      CSV를 1초마다 읽어 sensor_1~16 갱신
        ↓
② 위험률 산출      FastAPI에 보내 risk_score(0~100) 받기
        ↓
③ 상태 판단        정상 → 주의 → 위험 → 차단  (자동 전이)
        ↓
④ 자동 차단        밸브 닫기 → 경보 켜기 → 환기팬 켜기 (순차)
        ↓
⑤ 자동 복구        농도 안전해지면 → 밸브 다시 열고 경보 끄기
        ↓
   (다음 시나리오로 자동 전환) ──┘  ※ 전 과정 사람 개입 0
```

### 위험 등급 기준 (숫자로 약속)
| 등급 | risk_score | 색상 | 동작 |
|------|-----------|------|------|
| 정상 | < 40 | 🟢 초록 | 평상 운영 |
| 주의 | 40~59 | 🟡 노랑 | HUD 경고 |
| 위험 | 60~69 | 🔴 빨강 | 차단 카운트다운 |
| **차단** | **≥ 70** | ⬛ 회색 | **자동 차단 시퀀스 실행** |

> **threshold(차단 기준) = 70 으로 고정.** "LEL 기준 위험 시작점"이라는 근거가 있어 탐색하지 않습니다.

## 5. 꼭 알아야 할 용어 5개

| 용어 | 쉬운 설명 |
|------|-----------|
| **상태머신 (State Machine)** | "정상→주의→위험→차단→복구"처럼 **정해진 단계를 자동으로 옮겨가는 로직**. 신호등이 빨강→초록 자동으로 바뀌는 것과 같음 |
| **threshold (임계값)** | "여기를 넘으면 차단" 하는 **기준선**. 우리는 70으로 고정 |
| **차단 시퀀스** | 차단할 때 밸브→경보→환기팬을 **순서대로** 작동시키는 것 (한 번에 X) |
| **인터락 (Interlock)** | 안전 조건을 만족 못하면 **재개방을 막는 안전장치**. 환기가 충분히 안 됐으면 밸브를 다시 안 엶 |
| **히스테리시스 (Hysteresis)** | 차단(≥70)과 복구(<30) 기준을 **다르게** 둬서, 경계에서 밸브가 떨리는(차단↔복구 반복) 걸 막는 기법 |

## 7. 발표에서 자랑할 지표 (ML 점수 X)

기존엔 R²·MAE 같은 ML 지표였지만, 이제는 **자동화 지표**로 평가합니다:

- ⏱️ **차단 지연시간** — 위험 감지부터 밸브 닫힘까지 (목표 < 1초)
- ✅ **시퀀스 완결성** — 밸브·경보·환기 3개 모두 실행됐나
- 🔓 **자동 복구 성공률** — 안전해지면 무인으로 정상화됐나
- 🤖 **무인 데모 완주** — 사람 클릭 0번으로 끝까지 돌았나
