01 MAIN MENU
대시보드
시스템 전체 상태를 한눈에 확인할 수 있는 홈 화면입니다. 최근 시뮬레이션 결과, 최대 위험률, 자동차단 횟수 등 핵심 지표를 제공합니다.
주요 기능
  • 현재 시스템 상태 (대기 / 실행 중 / 차단완료)
  • 최근 5개 시뮬레이션 결과 요약
  • 자동화 지표 KPI (차단 지연·시퀀스 완결성·복구 성공률) 요약
  • 빠른 실행 — 마지막 시나리오로 바로 데모 진입
SCENARIO 01 · 데모 시나리오
발표용 자동 데모 실행
Actor: 발표자 / 강사
01
시스템 실행 및 대시보드 확인
02
연구소 시나리오 선택
03
[데모 실행] 버튼 클릭 → 자동 진행 시작
04
가스 누출 발생 → Particle System 활성화
05
위험률 상승 → HUD 갱신 확인
06
Threshold 초과 → 자동차단 발동
07
결과 요약 화면 확인 후 발표 설명
SCENARIO 02 · 정상 운영
위험률 낮음 — 자동차단 미발생
Actor: 연구원 / 운영자
01
시스템 실행 및 시나리오 선택
02
낮은 농도 CSV 데이터 재생
03
위험률이 threshold 미만으로 유지됨
04
HUD 상태 = 정상(GREEN) 유지
05
시뮬레이션 종료 → 결과에서 max_risk 낮음 확인
SCENARIO 03 · 위험 감지
위험률 상승 → 자동차단 발동
Actor: 연구원 / 시뮬레이션 테스터
01
고농도 가스 누출 시나리오 선택
02
누출 CSV 재생 → 위험률 급증
03
HUD: 정상 → 주의 → 위험 상태 변화
04
risk_score ≥ threshold → 자동차단 발동
05
밸브 CLOSED, 경보 ON, 환기팬 ON
06
차단 이벤트 DB 저장 확인
SCENARIO 04 · 결과 분석
시뮬레이션 결과 상세 분석
Actor: 강사 / 데이터 분석가
01
이력 화면에서 시뮬레이션 선택
02
max_risk, cutoff_time 확인
03
위험률 시계열 그래프 상세 보기
04
위험 요인 (농도/속도/지속시간) 분석 패널 확인
05
자동화 지표 (차단 지연·복구 성공률) 화면 확인
