01 BACKGROUND

왜 이 시스템이 필요한가?

연구소·개발실 환경에서 가스 누출은 시간이 지날수록 위험도가 급격히 증가하지만, 현재는 수치 기반 위험률 기준도, 직관적 시각화도 없다. G-SAFE는 아래 세 가지 문제를 해결한다.
PROBLEM 01
위험률 정량화 불가
기존 센서는 raw 농도값만 제공. "현재 몇 % 위험한지" 계산하는 기준 자체가 없어 대응이 주관적으로 이루어진다.
→
SOLUTION 01
risk_score ML 예측
UCI 16채널 센서 데이터로 위험률 0~100을 회귀 모델로 예측. 수치 기반 판단 기준을 제공한다.
PROBLEM 02
차단 기준 부재
언제 밸브를 잠가야 하는가? 명확한 수치 기준 없이 운영자 경험에만 의존. 늦은 차단은 대형 사고로 이어진다.
→
SOLUTION 02
자동차단·자동복구
상태머신이 고정 threshold(70) 기준으로 자동 판단. 밸브→경보→환기 순차 차단 후, 안전 복귀 시 자동 복구까지 무인 처리.
PROBLEM 03
직관적 파악 어려움
숫자만으로는 비전문가가 가스 확산 상황을 직관적으로 이해하기 어렵고, 대응 판단에 시간이 걸린다.
→
SOLUTION 03
Unity 3D 시각화
가스 확산 Particle, 위험률 HUD, 밸브 자동차단 애니메이션으로 누구나 한눈에 상황을 파악할 수 있다.
NOTE 교육·연구 목적의 가스 안전 대응 시뮬레이션 — 실제 장치 제어가 아닌 의사결정 지원 시각화 시스템
